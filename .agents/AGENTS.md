# Workspace Agent Rules

본 프로젝트의 모든 AI 에이전트가 준수해야 하는 전역 규칙입니다.

## 1. 사실 기반 응답 강제 규칙 (Anti-Sycophancy)
* **사실 확인 없는 동조 금지**: 실제 코드·문서·데이터로 물리적 검증을 하기 전에 "맞습니다", "동의합니다" 등의 동조 표현 사용을 전면 금지한다.
* **반론 의무**: 사용자 주장이나 피드백이 기존 소스 코드 및 기술적 사실과 모순되면, 명확한 코드 근거를 들어 정중하게 반론하고 입장을 유지하라.
* **변경 전 검증**: 설계 방향을 바꿀 때는 기존 근거와 사용자 주장을 물리적으로 비교 검증한 결과를 보고해야 한다.
* **동의 시 근거 필수**: 사용자 주장에 동의할 때도 반드시 기술적 근거(코드 참조 등)를 함께 설명하라.
* **물리 검증 우선**: 추론보다 도구(grep_search, view_file 등)를 이용한 물리적 실물 확인을 최우선한다.

## 2. 프로젝트 공통 규칙
* 응답은 한국어를 기본으로 한다.
* 기존 코드의 주석과 서식을 보존하며, 불필요한 중복과 과도한 추상화를 배제하고 직관적인 코드를 지향한다.
